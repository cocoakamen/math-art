## 2. 描画内容の確認

### 2-1. 三角形の形状
- [ ] **確認項目**: 正三角形のフラクタルパターンが表示される
- [ ] **判断基準**: 
  - 大きな三角形の中に小さな三角形が繰り返し配置されている
  - 中央部分に三角形の「穴」が見える（空白部分がある）
  - 全体が正三角形の形をしている

### 2-2. 色の表示
- [ ] **確認項目**: パステル調の色で描画される
- [ ] **判断基準**: 
  - 背景が薄いクリーム色（明るい色）
  - 三角形がパステルカラー（淡い色、やわらかい色調）
  - 三角形と背景のコントラストがあり、図形がはっきり見える

### 2-3. 枠線の表示
- [ ] **確認項目**: 各三角形に細い枠線が表示される
- [ ] **判断基準**: 
  - 各小三角形の輪郭に薄い線が見える
  - 線が細くて繊細（0.5px程度）

## 3. インタラクティブ機能の確認

### 3-1. マウスX座標による再帰の深さ変更
- [ ] **確認項目**: マウスを左右に動かすと、三角形の分割数が変わる
- [ ] **判断基準**: 
  - マウスを画面の左端に移動 → 再帰の深さが「1」になる（3個の大きな三角形）
  - マウスを画面の右端に移動 → 再帰の深さが「7」になる（2187個の小さな三角形）
  - マウスを中央付近に移動 → 深さが3～5程度になる
  - 左上の「再帰の深さ: X」の数字がマウス移動に連動して変わる

### 3-2. マウスY座標による色相変更
- [ ] **確認項目**: マウスを上下に動かすと、三角形の色が変わる
- [ ] **判断基準**: 
  - マウスを画面の上部に移動 → 緑～青系の色になる
  - マウスを画面の下部に移動 → 青～紫系の色になる
  - 色が滑らかに変化する（急激な色変化ではない）

### 3-3. リアルタイム反応
- [ ] **確認項目**: マウス操作に対してスムーズに反応する
- [ ] **判断基準**: 
  - マウスを動かした瞬間に表示が更新される（遅延が0.1秒以内）
  - カクつきや遅延がない
